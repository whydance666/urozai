# Урожай-Дорожай

ГИС-платформа для управления агробизнесом — мониторинг полей, земельный банк, агрономия, логистика и цепочки поставок.

---

## Стек

| Технология | Роль |
|---|---|
| HTML5 | разметка, семантическая структура |
| CSS3 | стили, анимации, адаптив |
| Vanilla JS | интерактивность без фреймворков |

---

## Структура проекта

```
/
├── index.html          # точка входа
├── css/
│   ├── main.css        # основные стили
│   └── fonts.css       # подключение шрифтов
├── JS/
│   └── main.js         # вся логика интерфейса
└── files/              # иконки и статика
```

---

## Что реализовано

**Сайдбар** — сворачивается/разворачивается по клику на логотип. В свёрнутом состоянии показывает только иконки, в открытом — иконки с подписями и сабменю. Закрывается кликом вне или по `Escape`.

**Сабменю** — раскрывается с анимацией через `max-height`. Открытие одного автоматически закрывает остальные.

**Поиск** — кнопка в шапке разворачивается в поле ввода. Закрывается кликом вне или по `Escape` если поле пустое.

**Адаптив** — три брейкпоинта: 1920px (дефолт), 1024px (планшет), 576px (мобилка).
